# Documentation Agent Instructions

This folder contains architectural and design documentation.

Maintain:
- High-level perspective
- Clear structure
- Neutral, professional tone
- Markdown suitable for long-term maintenance

Do not introduce implementation details unless explicitly requested.

Alt indhold i denne mappe skal skrives på dansk.

Krav:
- Professionelt og neutralt sprog
- Konsistent terminologi
- Ingen engelske overskrifter eller afsnit
- Indhold skal kunne læses af både IT og forretning

Engelske fagudtryk må anvendes, hvor de er almindeligt accepterede (fx API, system of record, role-based access), men teksten skal overordnet være på dansk.
